# LAIRM REFERENCE COMPENDIUM
## 28 Contextual and Paradigmatic Chapters

### Description

The Reference Compendium constitutes the contextual and paradigmatic corpus of LAIRM, structured in 4 parts containing 28 chapters. This compendium provides the historical context, sectoral dimensions, fundamental paradigms, and prospective visions.
