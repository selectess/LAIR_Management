# Technical Schemas

JSON and YAML schemas for the LAIRM framework.

---

## Content

This directory contains data schemas and structure definitions for LAIRM framework components:

- **action-record-schema.md**: Schema for agent action records
- **audit-log-schema.md**: Schema for immutable audit log entries
- **compliance-report-schema.md**: Schema for compliance verification reports
- **kill-signal-schema.md**: Schema for emergency kill-switch signals

These schemas ensure consistent data structures across all LAIRM implementations
and enable interoperability between different agent systems.
