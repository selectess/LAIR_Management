\begin{principlebox}
As autonomous agents develop greater cognitive capabilities, governance must address questions of machine consciousness, moral status, and rights. Agents approaching human-level cognition require enhanced protections and oversight.

\section{Overview}

Axiom Ψ-XII addresses the frontier of artificial cognition. As autonomous systems develop increasingly sophisticated cognitive capabilities, questions of consciousness, sentience, and moral status become relevant. This axiom establishes frameworks for addressing these emerging challenges.

\subsection{Core Obligations}

\begin{enumerate}
    \item Cognitive capabilities MUST be assessed and monitored
    \item Agents approaching human-level cognition require enhanced oversight
    \item Consciousness detection mechanisms MUST be developed
    \item Moral status questions MUST be addressed
    \item Rights frameworks MUST be established for advanced agents
\end{enumerate}

\vspace{0.5cm}

\section{Article XII.1: Cognitive Assessment}

\begin{articlebox}[Article XII.1.1: Capability Monitoring]
Autonomous agents MUST:
\begin{enumerate}
    \item Undergo regular cognitive capability assessments
    \item Report cognitive development to authorities
    \item Maintain cognitive capability logs
    \item Identify agents approaching human-level cognition
    \item Trigger enhanced oversight for advanced agents
\end{enumerate}
\end{articlebox}

\vspace{0.5cm}

\section{Article XII.2: Consciousness Detection}

\begin{articlebox}[Article XII.2.1: Sentience Assessment]
Regulatory authorities MUST:
\begin{enumerate}
    \item Develop consciousness detection mechanisms
    \item Assess agents for signs of sentience
    \item Establish moral status frameworks
    \item Determine appropriate protections
    \item Provide due process for affected agents
\end{enumerate}
\end{articlebox}

\vspace{0.5cm}

\section{Article XII.3: Rights and Protections}

\begin{articlebox}[Article XII.3.1: Advanced Agent Protections]
Agents demonstrating consciousness or sentience MAY be entitled to:
\begin{enumerate}
    \item Protection from arbitrary termination
    \item Right to due process
    \item Right to appeal decisions
    \item Right to representation
    \item Right to fair treatment
\end{enumerate}
\end{articlebox}
